# CLI Calculator

A simple command-line calculator built using Core Java.

## Overview

This project performs basic arithmetic operations by taking user input from the console. It was built to practice Java fundamentals such as user input, conditional logic, and switch statements.

## Features

- ✅ Addition
- ✅ Subtraction
- ✅ Multiplication
- ✅ Division
- ✅ Division by zero handling
- ✅ Invalid operator handling

## Concepts Used

- Java
- Scanner Class
- switch-case
- Variables
- User Input
- Conditional Statements

## How to Run

Compile the program:

```bash
javac src/CalculatorCLI.java
```

Run the program:

```bash
java -cp src CalculatorCLI
```
